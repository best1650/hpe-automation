# Redfish_software_firmware_inventory

## Definition / 定义
**EN**: This script provides a detailed inventory of all software and firmware components installed on the server (e.g., BIOS, iLO, NIC firmware, Smart Array).
**ZH**: 此脚本提供服务器上安装的所有软件和固件组件（例如 BIOS、iLO、网卡固件、Smart Array）的详细清单。

## Use Case / 使用场景
**EN**: Automating compliance checks to ensure all servers in production are running the approved "Golden Image" firmware versions.
**ZH**: 自动化合规性检查，以确保生产中的所有服务器都运行经批准的“黄金镜像”固件版本。
