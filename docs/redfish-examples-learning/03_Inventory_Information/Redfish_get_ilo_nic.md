# Redfish_get_ilo_nic

## Definition / 定义
**EN**: This script gathers detailed information about the iLO Network Interface Controller, including MAC addresses and current link status.
**ZH**: 此脚本收集有关 iLO 网络接口控制器 (NIC) 的详细信息，包括 MAC 地址和当前链路状态。

## Use Case / 使用场景
**EN**: Troubleshooting network connectivity issues by verifying the physical link status of the management port.
**ZH**: 通过验证管理端口的物理链路状态来排除网络连接问题。
