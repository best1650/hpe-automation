# Redfish_get_base_registry

## Definition / 定义
**EN**: This script retrieves the Base Message Registry, which provides human-readable messages for various events and error codes returned by the Redfish API.
**ZH**: 此脚本获取基础消息注册表，它为 Redfish API 返回的各种事件和错误代码提供人类可读的消息。

## Use Case / 使用场景
**EN**: Developing an alert monitoring tool that translates raw Redfish error codes into clear English/Chinese descriptions for system administrators.
**ZH**: 开发警报监控工具，将原始 Redfish 错误代码翻译为清晰的中/英文说明，供系统管理员查看。
