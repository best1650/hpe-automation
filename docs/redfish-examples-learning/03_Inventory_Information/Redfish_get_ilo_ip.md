# Redfish_get_ilo_ip

## Definition / 定义
**EN**: This script specifically retrieves the current IP addresses (IPv4/IPv6) configured for the iLO management interface.
**ZH**: 此脚本专门获取为 iLO 管理接口配置的当前 IP 地址（IPv4/IPv6）。

## Use Case / 使用场景
**EN**: Dynamic documentation of management IPs to ensure CMDB (Configuration Management Database) accuracy.
**ZH**: 管理 IP 的动态文档记录，以确保 CMDB（配置管理数据库）的准确性。
