# Redfish_get_LogicalDrives

## Definition / 定义
**EN**: This script identifies and lists all logical drives (RAID volumes) configured on the system's storage controllers.
**ZH**: 此脚本识别并列出系统存储控制器上配置的所有逻辑驱动器（RAID 卷）。

## Use Case / 使用场景
**EN**: Monitoring storage health by verifying that all expected logical volumes are online and report a healthy status.
**ZH**: 通过验证所有预期的逻辑卷是否在线并报告健康状态来监控存储健康。
