# Redfish_get_powermetrics_average

## Definition / 定义
**EN**: This script calculates and reports the average power consumption of the server over a specific period by querying the Redfish power metrics.
**ZH**: 此脚本通过查询 Redfish 功率指标，计算并报告服务器在特定时段内的平均功耗。

## Use Case / 使用场景
**EN**: Data center energy cost analysis and capacity planning by monitoring the actual power draw of different server models.
**ZH**: 通过监控不同服务器型号的实际功耗，进数据中心能源成本分析和容量规划。
