# Redfish_get_SmartArray_EncryptionSettings

## Definition / 定义
**EN**: This script gathers the encryption settings (e.g., whether local or external key management is used) for Smart Array controllers.
**ZH**: 此脚本收集 Smart Array 控制器的加密设置（例如，是使用本地还是外部密钥管理）。

## Use Case / 使用场景
**EN**: Inventorying security parameters to ensure that all sensitive data volumes are protected by the mandated encryption policy.
**ZH**: 盘点安全参数，以确保所有敏感数据卷都受到强制加密策略的保护。
