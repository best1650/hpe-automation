# Redfish_set_server_asset_tag

## Definition / 定义
**EN**: This script sets the user-defined Asset Tag for the server, which is stored in the BMC's non-volatile memory.
**ZH**: 此脚本为服务器设置用户定义的资产标签，该标签存储在 BMC 的非易失性存储器中。

## Use Case / 使用场景
**EN**: Digitally tagging each server with an internal asset tracking ID (e.g., "AST-2024-001") during the final stages of data center installation.
**ZH**: 在数据中心安装的最后阶段，使用内部资产跟踪 ID（例如“AST-2024-001”）为每台服务器进行数字化标记。
