# Redfish_computer_details

## Definition / 定义
**EN**: This script retrieves comprehensive details about the computer system, including model, serial number, processor information, memory capacity, and health status.
**ZH**: 此脚本获取有关计算机系统的全面详细信息，包括型号、序列号、处理器信息、内存容量和健康状态。

## Use Case / 使用场景
**EN**: Performing a baseline hardware inventory across thousands of nodes in a high-performance computing (HPC) cluster.
**ZH**: 在高性能计算 (HPC) 集群的数千个节点中执行基准硬件资产盘点。
