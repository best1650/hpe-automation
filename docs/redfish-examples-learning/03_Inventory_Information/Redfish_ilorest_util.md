# Redfish_ilorest_util

## Definition / 定义
**EN**: This is a utility script that provides general information about the iLO system, such as generation (e.g., Gen10) and basic versioning.
**ZH**: 这是一个实用程序脚本，提供有关 iLO 系统的常规信息，例如型号代次（例如 Gen10）和基础版本信息。

## Use Case / 使用场景
**EN**: Initial discovery phase of a management script to determine which generation-specific features are available on a target system.
**ZH**: 管理脚本的初始发现阶段，用以确定目标系统上有哪些特定代次的功能可用。
