# Redfish_get_resource_directory

## Definition / 定义
**EN**: This script retrieves the iLO Resource Directory, which is a map of all available Redfish URIs and resources supported by the specific iLO firmware.
**ZH**: 此脚本获取 iLO 资源目录，它是特定 iLO 固件支持的所有可用 Redfish URI 和资源的地图。

## Use Case / 使用场景
**EN**: Exploring the capabilities of a new iLO firmware version to identify new API endpoints for custom automation scripts.
**ZH**: 探索新 iLO 固件版本的功能，以识别用于自定义自动化脚本的新 API 端点。
