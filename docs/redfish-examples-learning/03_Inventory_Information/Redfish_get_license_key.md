# Redfish_get_license_key

## Definition / 定义
**EN**: This script retrieves the installed iLO license keys, which determine the available advanced management features.
**ZH**: 此脚本获取已安装的 iLO 许可证密钥，这些密钥决定了可用的高级管理功能。

## Use Case / 使用场景
**EN**: Auditing license compliance across a server fleet to ensure all units have the required "Advanced" or "Premium" features enabled.
**ZH**: 在服务器集群中审计许可证合规性，以确保所有单元都启用了所需的“高级”或“高级”功能。
