# Redfish_get_schema

## Definition / 定义
**EN**: This script downloads the JSON schema definitions from the BMC, which define the structure and data types of Redfish resources.
**ZH**: 此脚本从 BMC 下载 JSON 架构定义，这些定义确定了 Redfish 资源的结构和数据类型。

## Use Case / 使用场景
**EN**: Developing highly robust automation tools that validate the data received from the API against official schemas to handle version differences.
**ZH**: 开发高度健壮的自动化工具，这些工具会根据官方架构验证从 API 接收的数据，以处理版本差异。
