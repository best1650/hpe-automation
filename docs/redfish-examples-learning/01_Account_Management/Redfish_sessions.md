# Redfish_sessions

## Definition / 定义
**EN**: This script shows how to manually manage Redfish sessions, including creating a session and then properly logging out (deleting the session).
**ZH**: 此脚本展示了如何手动管理 Redfish 会话，包括创建会话然后正确登出（删除会话）。

## Use Case / 使用场景
**EN**: Building custom management tools where fine-grained control over session lifecycle is required to prevent "too many open sessions" errors.
**ZH**: 构建自定义管理工具时，需要对会话生命周期进行细粒度控制，以防止出现“打开会话过多”的错误。
