# Redfish_add_user_account

## Definition / 定义
**EN**: This script demonstrates how to add a new iLO Manager user account via the Redfish API. It allows you to specify the username, password, and permissions (Role ID) for the new user.
**ZH**: 此脚本演示了如何通过 Redfish API 添加新的 iLO 管理员用户账户。它允许您为新用户指定用户名、密码和权限（角色 ID）。

## Use Case / 使用场景
**EN**: Automated provisioning of new servers where a standard administrative account needs to be created across multiple systems for a specific IT team.
**ZH**: 在自动化安装新服务器的过程中，需要为特定的 IT 团队在多个系统中创建标准的管理员账户。
