# Redfish_modify_user_account

## Definition / 定义
**EN**: This script is used to modify the properties of an existing iLO user account, such as updating the password or changing the user's role/permissions.
**ZH**: 此脚本用于修改现有 iLO 用户账户的属性，例如更新密码或更改用户的角色/权限。

## Use Case / 使用场景
**EN**: Periodic security updates where administrative passwords need to be rotated across the entire server fleet.
**ZH**: 定期安全更新，需要在整个服务器集群中轮换管理员密码。
