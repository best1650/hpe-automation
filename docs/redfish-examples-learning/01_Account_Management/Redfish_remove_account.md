# Redfish_remove_account

## Definition / 定义
**EN**: This script demonstrates how to securely delete an existing user account from the iLO manager using Redfish.
**ZH**: 此脚本演示了如何使用 Redfish 安全地从 iLO 管理器中删除现有用户账户。

## Use Case / 使用场景
**EN**: Offboarding process for IT staff where their specific access accounts must be removed from all managed hardware.
**ZH**: IT 员工离职流程，必须从所有管理的硬件中删除其特定的访问账户。
