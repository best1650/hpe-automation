# Redfish_set_license_key

## Definition / 定义
**EN**: This script is used to install or activate a new iLO license key on the server, unlocking advanced features.
**ZH**: 此脚本用于在服务器上安装或激活新的 iLO 许可证密钥，以解锁高级功能。

## Use Case / 使用场景
**EN**: Automated activation of "iLO Advanced" licenses across a newly purchased shipment of servers to enable remote console and power management features.
**ZH**: 在新购买的一批服务器上自动激活“iLO Advanced”许可证，以启用远程控制台和电源管理功能。
