# Redfish_quickstart_redfish

## Definition / 定义
**EN**: This is a simple starter script that demonstrates how to initialize the Redfish client and perform a basic GET request to the Service Root.
**ZH**: 这是一个简单的入门脚本，演示了如何初始化 Redfish 客户端并对服务根目录执行基础的 GET 请求。

## Use Case / 使用场景
**EN**: Testing connectivity and authentication credentials with a new iLO management IP before building more complex automation scripts.
**ZH**: 在构建更复杂的自动化脚本之前，使用新的 iLO 管理 IP 测试连接和身份验证凭据。
