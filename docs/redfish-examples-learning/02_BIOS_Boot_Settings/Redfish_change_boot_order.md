# Redfish_change_boot_order

## Definition / 定义
**EN**: This script allows you to view and modify the persistent boot order of an HPE server using Redfish.
**ZH**: 此脚本允许您使用 Redfish 查看和修改 HPE 服务器的持久启动顺序。

## Use Case / 使用场景
**EN**: Reconfiguring a cluster of servers to boot from a new SAN (Storage Area Network) volume instead of local disks during a migration.
**ZH**: 在迁移过程中，将服务器集群重新配置为从新的 SAN（存储区域网络）卷而不是本地磁盘启动。
