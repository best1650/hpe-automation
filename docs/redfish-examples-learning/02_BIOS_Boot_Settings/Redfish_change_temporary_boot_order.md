# Redfish_change_temporary_boot_order

## Definition / 定义
**EN**: This script sets a one-time temporary boot override (e.g., boot from PXE on the next restart only).
**ZH**: 此脚本设置一次性临时启动覆盖（例如，仅在下次重启时从 PXE 启动）。

## Use Case / 使用场景
**EN**: Triggering a network boot for automated OS re-installation or firmware deployment without changing the permanent boot configuration.
**ZH**: 在不更改永久启动配置的情况下，触发网络启动以进行自动化操作系统重新安装或固件部署。
