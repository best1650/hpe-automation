# Redfish_bios_revert_default

## Definition / 定义
**EN**: This script resets all BIOS settings to their factory default values using the Redfish interface.
**ZH**: 此脚本使用 Redfish 接口将所有 BIOS 设置重置为出厂默认值。

## Use Case / 使用场景
**EN**: Recovering a server from a misconfigured state where custom BIOS settings are causing stability or boot issues.
**ZH**: 当自定义 BIOS 设置导致稳定性或启动问题时，将服务器从错误配置状态中恢复。
