# Redfish_set_bios_iscsi

## Definition / 定义
**EN**: This script provides an automated way to configure iSCSI boot parameters within the BIOS settings through Redfish.
**ZH**: 此脚本提供了通过 Redfish 在 BIOS 设置中配置 iSCSI 启动参数的自动化方法。

## Use Case / 使用场景
**EN**: Automating the "Boot from SAN" configuration for diskless servers in a high-density compute environment.
**ZH**: 为高密度计算环境中的无盘服务器自动化配置“从 SAN 启动”。
