# Redfish_change_bios_setting

## Definition / 定义
**EN**: This script demonstrates how to change specific BIOS settings (e.g., Enable/Disable Virtualization technology) through the Redfish API.
**ZH**: 此脚本演示了如何通过 Redfish API 更改特定的 BIOS 设置（例如启用/禁用虚拟化技术）。

## Use Case / 使用场景
**EN**: Remotely optimizing server BIOS for specific workloads (e.g., enabling High Performance mode for a database server) without physical intervention.
**ZH**: 无需物理干预，即可远程优化服务器 BIOS 以适应特定工作负载（例如为数据库服务器启用高性能模式）。
