# Redfish_enable_secure_boot

## Definition / 定义
**EN**: This script simplifies the process of enabling the UEFI Secure Boot feature on an HPE iLO system via Redfish.
**ZH**: 此脚本简化了通过 Redfish 在 HPE iLO 系统上启用 UEFI 安全启动功能的过程。

## Use Case / 使用场景
**EN**: Enforcing security compliance across a data center by ensuring all servers have Secure Boot enabled to prevent rootkits and unauthorized boot loaders.
**ZH**: 通过确保所有服务器都启用了安全启动来防止 Rootkit 和未经授权的引导加载程序，从而在数据中心强制执行安全合规性。
