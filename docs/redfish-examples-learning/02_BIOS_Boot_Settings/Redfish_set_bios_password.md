# Redfish_set_bios_password

## Definition / 定义
**EN**: This script allows administrators to set or change the BIOS administrator password via the Redfish API.
**ZH**: 此脚本允许管理员通过 Redfish API 设置或更改 BIOS 管理员密码。

## Use Case / 使用场景
**EN**: Hardening server security by setting a BIOS password across all hardware to prevent unauthorized local BIOS modifications.
**ZH**: 通过在所有硬件上设置 BIOS 密码来加强服务器安全，防止未经授权的本地 BIOS 修改。
