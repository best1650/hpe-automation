# Redfish_configure_snmp

## Definition / 定义
**EN**: This script configures Simple Network Management Protocol (SNMP) settings on the iLO, including community strings and trap destinations.
**ZH**: 此脚本在 iLO 上配置简单网络管理协议 (SNMP) 设置，包括团体字符串和陷阱目标。

## Use Case / 使用场景
**EN**: Integrating new servers into a centralized monitoring system like Nagios or Zabbix to receive hardware health alerts via SNMP.
**ZH**: 将新服务器集成到 Nagios 或 Zabbix 等集中监控系统中，以通过 SNMP 接收硬件健康警报。
