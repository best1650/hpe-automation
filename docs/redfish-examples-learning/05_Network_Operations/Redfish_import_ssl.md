# Redfish_import_ssl

## Definition / 定义
**EN**: This script is used to import a signed SSL certificate (and optionally its private key) into the iLO manager.
**ZH**: 此脚本用于将已签名的 SSL 证书（以及可选的私钥）导入 iLO 管理器。

## Use Case / 使用场景
**EN**: Completing the certificate automation loop by uploading the signed certificate from the CA back to the server.
**ZH**: 通过将 CA 签名的证书重新上传到服务器，完成证书自动化循环。
