# Redfish_mount_virtual_media_iso

## Definition / 定义
**EN**: This script allows you to remotely mount an ISO image as a virtual CD/DVD drive on the server via an HTTP/HTTPS URL.
**ZH**: 此脚本允许您通过 HTTP/HTTPS URL 将 ISO 镜像作为虚拟 CD/DVD 驱动器远程挂载到服务器上。

## Use Case / 使用场景
**EN**: Remotely booting a server into a live diagnostic tool or an OS installer without needing physical access to the server room.
**ZH**: 无需进入物理机房，即可远程将服务器引导至实时诊断工具或操作系统安装程序。
