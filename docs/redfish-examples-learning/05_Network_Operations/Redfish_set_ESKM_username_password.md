# Redfish_set_ESKM_username_password

## Definition / 定义
**EN**: This script allows you to configure the authentication credentials (username and password) that iLO uses to connect to the external ESKM server.
**ZH**: 此脚本允许您配置 iLO 用于连接外部 ESKM 服务器的身份验证凭据（用户名和密码）。

## Use Case / 使用场景
**EN**: Updating the service account credentials used for key management across all hardware following a corporate security policy update.
**ZH**: 在公司安全策略更新后，更新用于所有硬件密钥服务的服务帐户凭据。
