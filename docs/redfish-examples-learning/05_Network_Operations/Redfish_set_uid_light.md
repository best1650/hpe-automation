# Redfish_set_uid_light

## Definition / 定义
**EN**: This script allows you to turn the Unit Identification (UID) blue light on or off via Redfish.
**ZH**: 此脚本允许您通过 Redfish 打开或关闭单元标识 (UID) 蓝色指示灯。

## Use Case / 使用场景
**EN**: Helping a remote technician physically locate a specific server in a crowded data center aisle by flashing its UID light.
**ZH**: 通过闪烁 UID 指示灯，帮助现场技术人员在拥挤的数据中心机柜中物理定位特定的服务器。
