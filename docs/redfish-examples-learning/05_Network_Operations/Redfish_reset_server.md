# Redfish_reset_server

## Definition / 定义
**EN**: This script demonstrates how to perform various types of server resets (e.g., Force Restart, Power Cycle) via the Redfish API.
**ZH**: 此脚本演示了如何通过 Redfish API 执行各种类型的服务器重置（例如强制重启、电源周期）。

## Use Case / 使用场景
**EN**: Remotely force-restarting a hung server that is unresponsive to serial-over-LAN or other OS-level management tools.
**ZH**: 远程强制重启对串口 LAN 或其他操作系统级管理工具无响应的死机服务器。
