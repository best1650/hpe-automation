# Redfish_reboot_server

## Definition / 定义
**EN**: This script initiates a soft or hard reboot of the server system through the Redfish interface.
**ZH**: 此脚本通过 Redfish 接口启动服务器系统的软重启或硬重启。

## Use Case / 使用场景
**EN**: Orchestrating a controlled reboot of multiple servers after a software update or configuration change that requires a system restart.
**ZH**: 在需要重启系统的软件更新或配置更改后，对多台服务器进行受控重启编排。
