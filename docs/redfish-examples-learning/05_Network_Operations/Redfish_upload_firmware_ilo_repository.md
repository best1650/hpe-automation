# Redfish_upload_firmware_ilo_repository

## Definition / 定义
**EN**: This script uploads a firmware component (e.g., a .bin or .exe file) to the iLO Repository, allowing it to be flashed later.
**ZH**: 此脚本将固件组件（例如 .bin 或 .exe 文件）上传到 iLO 存储库，以便稍后进行闪存。

## Use Case / 使用场景
**EN**: Staging firmware updates for a future maintenance window by pre-uploading all required components to the server's local repository.
**ZH**: 通过将所有必需组件预上传到服务器的本地存储库，为未来的维护窗口暂存固件更新。
