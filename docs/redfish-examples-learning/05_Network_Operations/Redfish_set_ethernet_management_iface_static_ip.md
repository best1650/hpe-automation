# Redfish_set_ethernet_management_iface_static_ip

## Definition / 定义
**EN**: This script changes the iLO management interface from DHCP to a fixed static IP address.
**ZH**: 此脚本将 iLO 管理接口从 DHCP 更改为固定的静态 IP 地址。

## Use Case / 使用场景
**EN**: Transitioning a server from the "staging" network (DHCP) to its permanent production location with a strictly assigned static management IP.
**ZH**: 在生产环境中，将服务器从“暂存”网络（DHCP）迁移到具有严格分配的静态管理 IP 的永久位置。
