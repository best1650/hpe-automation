# Redfish_set_ESKM_PrimaryKeyServer

## Definition / 定义
**EN**: This script configures the primary and secondary ESKM server addresses that iLO should use for key management.
**ZH**: 此脚本配置 iLO 用于密钥管理的主备 ESKM 服务器地址。

## Use Case / 使用场景
**EN**: Automating the rollout of new Enterprise Key Management infrastructure by updating the server pointers on all managed hardware.
**ZH**: 通过更新所有管理硬件上的服务器指针，实现在新企业密钥管理基础设施的自动化部署。
