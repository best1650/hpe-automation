# Redfish_update_ilo_firmware

## Definition / 定义
**EN**: This script automates the iLO firmware update process by providing a URL to the firmware image or using a local file.
**ZH**: 此脚本通过提供指向固件镜像的 URL 或使用本地文件，实现 iLO 固件更新过程的自动化。

## Use Case / 使用场景
**EN**: Mass-deployment of a critical security patch for iLO firmware across an entire server fleet to mitigate newly discovered vulnerabilities.
**ZH**: 在整个服务器集群中大规模部署 iLO 固件的关键安全补丁，以减轻新发现的漏洞。
