# Redfish_set_snmp_alert

## Definition / 定义
**EN**: This script configures specific alert types and destinations for SNMP traps, ensuring specific events trigger notifications.
**ZH**: 此脚本配置 SNMP 陷阱的特定警报类型和目标，确保特定事件触发通知。

## Use Case / 使用场景
**EN**: High-priority alert routing, such as ensuring power supply failures are sent immediately to a critical infrastructure alert pager.
**ZH**: 高优先级警报路由，例如确保电源故障立即发送到关键基础设施警报传呼机。
