# Redfish_set_ilo_ntp_servers

## Definition / 定义
**EN**: This script specifically allows administrators to update the list of NTP servers that iLO uses for time synchronization without enabling/disabling the service.
**ZH**: 此脚本专门允许管理员更新 iLO 用于时间同步的 NTP 服务器列表，而无需启用/禁用该服务。

## Use Case / 使用场景
**EN**: Migrating to a newer set of corporate NTP servers and updating all managed hardware to ensure continued time accuracy.
**ZH**: 迁移到新的公司 NTP 服务器组，并更新所有管理的硬件以确保持续的时间准确性。
