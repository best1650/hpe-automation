# Redfish_clear_ahs_data

## Definition / 定义
**EN**: This script initiates the clearing of the Active Health System (AHS) logs from the iLO manager.
**ZH**: 此脚本启动从 iLO 管理器清除活动健康系统 (AHS) 日志的操作。

## Use Case / 使用场景
**EN**: Maintenance procedure where old health logs are wiped to prepare for a fresh troubleshooting session or after a successful hardware repair.
**ZH**: 维护程序，清空旧的健康日志，为新的故障排除会话做准备，或在成功的硬件维修后执行。
