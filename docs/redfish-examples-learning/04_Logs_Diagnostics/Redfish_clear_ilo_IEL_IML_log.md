# Redfish_clear_ilo_IEL_IML_log

## Definition / 定义
**EN**: This script clears the iLO Event Log (IEL) and the Integrated Management Log (IML), which record critical system events and warnings.
**ZH**: 此脚本清除 iLO 事件日志 (IEL) 和集成管理日志 (IML)，这些日志记录了关键的系统事件和警告。

## Use Case / 使用场景
**EN**: Clearing warning messages from the dashboard after the underlying hardware issue has been resolved to return the health lights to green.
**ZH**: 在潜在硬件问题解决后，清除仪表板上的警告消息，使健康指示灯恢复为绿色。
