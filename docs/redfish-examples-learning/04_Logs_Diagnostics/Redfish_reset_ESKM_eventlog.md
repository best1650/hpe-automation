# Redfish_reset_ESKM_eventlog

## Definition / 定义
**EN**: This script clears the specific event logs related to ESKM interactions, allowing for a fresh start during security audits.
**ZH**: 此脚本清除与 ESKM 交互相关的特定事件日志，以便在安全审计期间重新开始记录。

## Use Case / 使用场景
**EN**: Troubleshooting encryption key retrieval errors by clearing old logs to isolate new connection attempts to the ESKM server.
**ZH**: 通过清除旧日志来隔离与 ESKM 服务器的新连接尝试，从而排除加密密钥获取错误。
