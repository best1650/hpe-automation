# Redfish_get_ahs_data

## Definition / 定义
**EN**: This script automates the downloading of the Active Health System (AHS) telemetry data, which is critical for troubleshooting hardware failures.
**ZH**: 此脚本自动化下载活动健康系统 (AHS) 遥测数据，这对于排除硬件故障至关重要。

## Use Case / 使用场景
**EN**: Collecting a "Health Report" automatically when a server reports a hardware warning to attach to a support case for HPE service.
**ZH**: 当服务器报告硬件警告时，自动收集“健康报告”，以便将其附加到 HPE 服务支持案例中。
