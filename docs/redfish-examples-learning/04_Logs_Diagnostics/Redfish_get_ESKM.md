# Redfish_get_ESKM

## Definition / 定义
**EN**: This script retrieves configuration details for Enterprise Secure Key Manager (ESKM) integration, which is used for managing encryption keys.
**ZH**: 此脚本获取企业安全密钥管理器 (ESKM) 集成的配置详情，该集成用于管理加密密钥。

## Use Case / 使用场景
**EN**: Verifying the configuration of external key management servers to ensure data-at-rest encryption is functioning correctly across the storage subsystem.
**ZH**: 验证外部密钥管理服务器的配置，以确保存储子系统的静态数据加密功能运行正常。
